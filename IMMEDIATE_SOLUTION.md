# 즉시 해결 방법

## 🚀 지금 바로 해보세요 (10초)

### 브라우저 콘솔에서 실행

1. **개발자 도구 열기** (F12)
2. **Console 탭 선택**
3. **다음 코드 복사해서 붙여넣기:**

```javascript
window.location.href = 'https://optlisting-production.up.railway.app/api/ebay/auth/start?user_id=default-user'
```

4. **Enter 키 누르기**

→ **바로 eBay 로그인 페이지로 이동합니다!**

## ✅ 이렇게 하면

1. eBay 로그인 페이지로 이동
2. eBay 로그인 및 권한 승인
3. Railway가 콜백 받아서 토큰 저장
4. Dashboard로 리다이렉트
5. **연결 완료!**
