# Vercel GitHub 권한 설정 단계별 가이드

## 🎯 가장 간단한 방법 (Vercel에서)

### 1단계: Vercel 접속
- https://vercel.com
- 로그인

### 2단계: 새 프로젝트 추가
- "Add New Project" 버튼 클릭

### 3단계: GitHub 연결
- "Continue with GitHub" 클릭
- GitHub 로그인
- **권한 승인** (자동으로 요청됨)

### 4단계: 저장소 선택
- `optlisting-team/optlisting-` 찾아서 선택

---

## 📍 GitHub Settings에서 확인하는 방법

### 경로:
```
GitHub → 우측 상단 프로필 → Settings → Applications → Installed GitHub Apps → Vercel
```

### 확인할 것:
1. Vercel 앱이 설치되어 있는지
2. `optlisting-team` 조직에 접근 권한이 있는지
3. `optlisting-` 저장소가 목록에 있는지

---

## 🔧 문제 해결

### 조직 저장소가 안 보일 때:
1. Vercel → "Add New Project"
2. "Configure" 또는 "Grant access" 클릭
3. `optlisting-team` 조직 선택

---

**가장 쉽게: Vercel에서 "Add New Project" 클릭하면 자동으로 권한 요청 나옵니다!** ✅
