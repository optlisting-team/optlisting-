# OptListing Frontend

React + Vite + Tailwind CSS frontend for OptListing SaaS application.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`

## Build

```bash
npm run build
```

## Features

- Dashboard with zombie listing summary
- Table view of all zombie listings
- Source badges (Amazon, Walmart, Unknown)
- CSV export for AutoDS, Yaballe, and eBay File Exchange
