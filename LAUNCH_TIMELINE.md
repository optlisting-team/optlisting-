# ⏱️ OptListing 런칭 타임라인 예상

## 🎯 핵심 작업별 예상 소요 시간

### 1. 결제 시스템 테스트 (Lemon Squeezy) - **30분~1시간**

**작업 내용:**
- Test Mode 활성화 확인: **5분**
- 테스트 상품 생성: **10분**
- Product ID를 프론트엔드에 반영: **5분**
- 테스트 결제 및 웹훅 확인: **10분**
- 크레딧 추가 확인: **5분**

**총 예상 시간: 30분~1시간**

---

### 2. 로그인/인증 테스트 (Supabase) - **15~30분**

**작업 내용:**
- Google OAuth 플로우 테스트: **10분**
- 세션 유지 확인: **5분**
- 로그아웃 테스트: **5분**

**총 예상 시간: 15~30분**

---

### 3. eBay API 실제 연동 테스트 - **1~2시간**

**작업 내용:**
- OAuth 플로우 테스트: **20분**
  - Settings에서 "Connect eBay" 클릭
  - eBay 승인 완료
  - 토큰 DB 저장 확인
- 실제 리스팅 가져오기: **20분**
  - API 호출
  - 데이터 수신 및 DB 저장 확인
  - 프론트엔드 표시 확인
- 토큰 자동 갱신 확인: **10분**
  - Worker 로그 확인
- Webhook 삭제 로직 테스트: **30분**
  - 테스트 웹훅 전송
  - Challenge-Response 검증
  - 삭제 로직 실행 확인

**총 예상 시간: 1~2시간** (eBay API 응답 시간에 따라 변동 가능)

---

### 4. 공급처 CSV 추출 테스트 - **30분~1시간**

**작업 내용:**
- 각 공급처 포맷 테스트 (6개): **20분**
  - AutoDS, Yaballe, eBay, Wholesale2B, Shopify Matrixify, Shopify Tagging
- Shopify 경유 항목 분리 테스트: **10분**
- CSV 파일 형식 확인: **10분**

**총 예상 시간: 30분~1시간**

---

### 5. DEMO_MODE 비활성화 - **10분**

**작업 내용:**
- `Dashboard.jsx`에서 `DEMO_MODE = false`: **2분**
- 더미 데이터 생성 코드 주석 처리: **5분**
- 테스트: **3분**

**총 예상 시간: 10분**

---

### 6. 환경 변수 최종 확인 - **15분**

**작업 내용:**
- Railway 환경 변수 확인: **5분**
- Vercel 환경 변수 확인: **5분**
- 각 변수 값 검증: **5분**

**총 예상 시간: 15분**

---

### 7. 보안 및 에러 핸들링 확인 - **20분**

**작업 내용:**
- CORS 설정 확인: **5분**
- API 키 보안 확인: **5분**
- 에러 처리 테스트: **10분**

**총 예상 시간: 20분**

---

## 💡 시간 절약 팁

1. **병렬 작업**
   - 결제 테스트와 로그인 테스트는 동시에 가능
   - CSV 테스트는 다른 작업 중에 백그라운드로 진행 가능

2. **우선순위**
   - 필수 기능부터 테스트 (결제, 로그인, eBay API)
   - 선택적 기능은 런칭 후 보완

3. **자동화**
   - 스크립트로 반복 테스트 자동화
   - CI/CD 파이프라인 구축 (선택사항)
