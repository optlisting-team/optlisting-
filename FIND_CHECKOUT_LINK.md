# Lemon Squeezy 체크아웃 링크 찾는 방법

## Step 1: 제품 클릭

1. 현재 화면에서 **"Credit Pack_1000"** 제품명을 클릭
2. 제품 상세 페이지로 이동

## Step 2: Variants 탭 확인

1. 상세 페이지 상단에 **"Variants"** 탭 클릭
2. Variant 목록 확인

## Step 3: Variant 클릭

1. Variant를 클릭 (보통 "Default" 또는 Variant 이름)
2. Variant 상세 페이지로 이동

## Step 4: 체크아웃 링크 확인

Variant 상세 페이지에서 찾아야 할 것들:

1. **"Share"** 버튼 (보통 우측 상단)
2. **"Get checkout link"** 버튼
3. **"Preview"** 버튼
4. 또는 페이지에 표시된 **체크아웃 URL**

---

## 대안: URL에서 직접 확인

브라우저 주소창을 확인하세요:

- 제품 상세 페이지: `/products/[PRODUCT_ID]`
- Variant 상세 페이지: `/products/[PRODUCT_ID]/variants/[VARIANT_ID]`

Variant ID가 URL에 표시되어 있을 수 있습니다.

---

## 빠른 확인 방법

1. **Credit Pack_1000** 제품명 클릭
2. 페이지가 로드되면 **브라우저 주소창 확인**
3. URL에서 숫자 확인 (Product ID 또는 Variant ID)
4. Variants 탭 클릭
5. Variant 클릭 후 주소창 다시 확인
